# Test Readiness Tracker

Cloud-based driving school management system built with AWS serverless architecture.

## 🚀 Live Demo
[View Live App](https://naheem1.github.io/test-progress-tracker/index.html)

## 🛠️ Technologies
- **Backend:** AWS Lambda, API Gateway, DynamoDB
- **Frontend:** HTML, CSS, JavaScript
- **Infrastructure:** Terraform

## 📋 Features
- Real-time student progress tracking
- DVSA compliance (9 required maneuvers)
- Test readiness calculator
- Mobile responsive design

## 💰 Cost
< $1/month to run

## 📦 Deployment
See `terraform/` folder for infrastructure code.
